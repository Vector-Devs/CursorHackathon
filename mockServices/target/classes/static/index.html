Testable APIs Reference
POST /api/vessels_operations/get-vessels-by-area
Fetch mock vessels filtered by proximity using Haversine calculation. Speed scales down in War Areas.
URL Parameters:
latitude longitude circle_radius (km)
POST /api/vessels_operations/get-places
Fetch places (Ports, Cities, Countries, Chokepoints) within 100 NM (185.2 km) radius.
URL Parameters:
latitude longitude
GET /api/v1/article/getArticles
Fetch mock intelligence articles (Maritime Security Bulletins).
Fallback Support:
Returns long articles structure (Conflict/Piracy items).
All POST endpoints consume standard Form URL-encoded fields in parameters.
